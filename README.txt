## Instruction
To run the web application:  
. Install Vagrant and Virtual Box   
. Launch the Vagrant VM (by typing `vagrant up` in the directory */category* from the terminal).  
. From directory */category*, initialize the application database by typing `python database_setup.py` follows by `python movies.py`.  
. From directory */category*, run the application within the VM by typing `python catalogue.py` into the Terminal.  
. Access the application by visiting http://localhost:8000 locally on the browser.
